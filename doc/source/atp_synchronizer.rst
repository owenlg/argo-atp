ATP Synchronizer
================

This is a python based package which periodically synchronize data from various topology providers. It reads topology related information from diffeernt sources and write to Oracle and MySQL backends. It includes all scripts for Oracle and MYSQL databases. It contains the following modules:
 * Input (:py:mod:`atp_synchronizer.input`)
 * Output (:py:mod:`atp_synchronizer.output`)
 * Config (:py:mod:`atp_synchronizer.config`)
 * ATP Synchro (:py:mod:`atp_synchronizer.atp_synchro`)
